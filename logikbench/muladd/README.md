Unsigned Integer Multiply and Add
============================================

## Description

Parametrized integer multiply-and-add that takes two unsigned N-bit wide inputs and adds it to a third M bit wide input to returns an M-bit wide result.

## Parameters

- N: input data width
- M: output data width
